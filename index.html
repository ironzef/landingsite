What Will You Learn In This Masterclass?

Server and Backend

Learn about HTTP and REST APIs and understand how apps communicate with their backend by tweeting from a Linux Terminal using Twitter APIs.

Data analysis

Learn to use Linux and get familiar with commonly used developer workflows by performing data analysis using Linux.

Cloud Devlopement

Get introduced to Cloud Deployment by actually deploying a real web app to a cloud-based AWS server.

Html basic to advance

Master HTML elements, tags, and attributes by using them to structure the web pages of QTrip.

CSS basic to advance

Learn CSS and Flexbox by utilizing them to enhance the layout and position of various elements on the web page.

Logic Building by JavaScript

Learn to deal with the business logic, security, performance, scalability, and handling request-response of the application, design the core application workflows and use technologies like JavaScript, Python, Java, and .NET.
